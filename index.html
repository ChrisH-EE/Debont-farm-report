核心结论
这是黑龙江省甘南县一个生长季的农机作业日志，主体为金丰公社服务机队，另有保农农场。 每条记录是带机手、设备、面积、机具幅宽的 GPS 轨迹，不是地块权属图。 除季节流水线外，更深层的信号是：季中管护（深松/植保）强烈预测同地收获覆盖，以及 幅宽几乎可唯一判定作业类型。

更深发现二：负荷集中、空窗期与组织分工

在季节流水线与幅宽指纹之外，逐条重算轨迹后浮现的三类更深模式：作业量高度集中在少数设备、 生长期存在一段完全静默的空窗、以及两家组织在作业结构与人员绑定上的系统性差异。

面积与幅宽实际含义

面积 ≈ 0.8 × 轨迹长度 × 幅宽

面积与 GPS 点数相关 0.86–0.96；相对「轨迹×幅宽」的中位比值约 0.8 （重叠/效率折减）。面积是作业覆盖量，不是地籍亩数。

幅宽 ≈ 机具类型

播种 1.3–4.4 米；深松多为 3.25/3.3 米；收获 3.25–5.85 米；喷雾 7.8–23.4 米。 宽幅几乎只做植保。

实用「若 → 则」速查

若 1km 网格内有播种又有深松 → 则 同网格很可能有收获（≈90%，无深松时≈43%）。
若 网格有深松或植保 → 则 几乎必有先前播种（≈90%）。
若 幅宽 ≥ 7 米 → 则 作业一定是植保。
若 幅宽为 1.3 / 1.95 / 2.6 米 → 则 作业一定是播种。
若 同一编号跨播种与收获 → 则 挂具幅宽已更换（14/14）。
若 月份在 4–5 月 → 则 作业几乎一定是播种。
若 月份在 6–7 月 → 则 作业为植保或深松。
若 月份在 9–11 月 → 则 作业为收获。
若 播种后约 30 天 → 则 同设备同乡镇进入深松/植保窗口。
若 播种后约 5 个月 → 则 同设备同乡镇进入收获（中位 152 天）。
若 乡镇有植保 → 则 该乡镇具备播种+深松+收获全流程。
若 机手不是张志坚 → 则 整季只绑定一台设备。
若 同设备收/播面积比 > 3 → 则 视为外向型代收为主。
若 乡镇仅有收获 → 则 视为代收/跨区服务目的地。
若 日期落在 8 月1日–9月25日 → 则 档案中无任何作业记录（56天生长期空窗）。
若 设备属于全季面积前 5 名（共20台）→ 则 这 5 台合计完成全季 54% 的作业面积。
若 组织为金丰公社 → 则 植保面积占比≈32%，约为保农农场（17%）的两倍。
若 同设备同作业类型 → 则 幅宽整季几乎恒定（54/56 组仅一个档位）。
若 机手确定 → 则 所属组织唯一，两家组织人员互不交叉（11人 11机构，0 交叉）。
若 乡镇轨迹跨度仅几公里 → 则 多为纯收获代收点；跨度达十余到三十余公里 → 为多地块拼合的全流程服务片区。
注意事项
以上为观测关联（含时间顺序与空间共现），不是实验室因果。 共有设备上收获上报面积约为播种的 2.4 倍；空间匹配约 76%–84%，但不是地块 ID 一一对应—— 只有轨迹中心点。3.25 / 3.9 / 5.2 米幅宽需结合月份区分深松与收获。
