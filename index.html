核心结论 / Core takeaway
这是黑龙江省甘南县一个生长季的农机作业日志，主体为金丰公社服务机队，另有保农农场。 每条记录是带机手、设备、面积、机具幅宽的 GPS 轨迹，不是地块权属图。 除季节流水线外，更深层的信号是：季中管护（深松/植保）强烈预测同地收获覆盖，以及 幅宽几乎可唯一判定作业类型。 One growing-season machinery log for Gannan County, Heilongjiang — mainly the Jinfeng Commune service fleet plus Baonong Farm. Each record is a GPS track with operator, device, area, and implement width — not a parcel cadastre. Beyond the seasonal pipeline, deeper signals are: mid-season care (deep tillage / plant protection) strongly predicts local harvest coverage, and WIDTH nearly uniquely classifies operation type.

反向：更强Reverse: even stronger

若网格有深松 → 约 90% 已有播种；有植保 → 约 91% 已有播种。 季中作业几乎不会落在从未播种的条带上。 If a cell has deep tillage → ~90% already has seeding; plant protection → ~91%. Mid-season work almost never lands on never-seeded strips.

意外：深松在播种之后 / Surprise: deep tillage after seeding
常见预期是「深松→播种」（播前整地）。数据里播种多在 5 月结束，深松在 6–7 月—— 这里是季中中耕/出苗后深松。植保与深松在 6 月重叠，不是严格先后。 Naive expectation is deep tillage → seeding (pre-plant prep). In the data, seeding mostly ends in May, deep tillage runs Jun–Jul — mid-season / post-emergence loosening. Spray and deep tillage overlap in June; not a strict before/after pair.

更深发现二：负荷集中、空窗期与组织分工 Deeper round 2: load concentration, silent window & org roles

在季节流水线与幅宽指纹之外，逐条重算轨迹后浮现的三类更深模式：作业量高度集中在少数设备、 生长期存在一段完全静默的空窗、以及两家组织在作业结构与人员绑定上的系统性差异。 Beyond the seasonal pipeline and width fingerprint, three deeper patterns: workload concentrated on a few devices, a fully silent growth-season window, and systematic org differences in job mix and staff binding.

意外：56 天完全静默的生长期空窗 / Surprise: 56-day silent growth window
逐日统计 5,802 条记录后，8月1日至9月25日（连续 56 天）没有任何一条作业记录—— 横跨全部四类作业、全部 20 台设备。这正是深松/植保收尾（约7月22日）到收获开始（9月26日） 之间的作物生长期：档案系统在这段窗口里完全「沉默」，说明数据只记录机具下地作业，不含巡田等其他活动。 After daily tally of all 5,802 jobs, Aug 1 – Sep 25 (56 consecutive days) has zero records — across all four ops and all 20 devices. This is the crop growth window between mid-season work ending (~Jul 22) and harvest starting (Sep 26). The archive is fully silent here: it logs machine field work only, not scouting or other activities.

面积与幅宽实际含义 What AREA and WIDTH actually mean

面积 ≈ 0.8 × 轨迹长度 × 幅宽AREA ≈ 0.8 × path length × WIDTH

面积与 GPS 点数相关 0.86–0.96；相对「轨迹×幅宽」的中位比值约 0.8 （重叠/效率折减）。面积是作业覆盖量，不是地籍亩数。 AREA correlates 0.86–0.96 with GPS point count; median ratio vs path×width ≈ 0.8 (overlap / efficiency). AREA is work coverage, not cadastre acreage.

幅宽 ≈ 机具类型WIDTH ≈ implement class

播种 1.3–4.4 米；深松多为 3.25/3.3 米；收获 3.25–5.85 米；喷雾 7.8–23.4 米。 宽幅几乎只做植保。 Seeders 1.3–4.4 m; deep tillage mostly 3.25/3.3 m; harvesters 3.25–5.85 m; spray 7.8–23.4 m. Wide implements are almost only plant protection.

实用「若 → 则」速查 Practical if → then quick reference

若 / If 1km 网格内有播种又有深松 → 则 / then 同网格很可能有收获（≈90%，无深松时≈43%）。Cell has seed+deep → likely harvest (~90%; ~43% without deep).
若 / If 网格有深松或植保 → 则 / then 几乎必有先前播种（≈90%）。Cell has deep/spray → prior seeding almost sure (~90%).
若 / If 幅宽 ≥ 7 米 → 则 / then 作业一定是植保。WIDTH ≥ 7 m → plant protection (deterministic).
若 / If 幅宽为 1.3 / 1.95 / 2.6 米 → 则 / then 作业一定是播种。WIDTH ∈ {1.3, 1.95, 2.6} → seeding.
若 / If 同一编号跨播种与收获 → 则 / then 挂具幅宽已更换（14/14）。Same SN across seed & harvest → WIDTH changed (14/14).
若 / If 月份在 4–5 月 → 则 / then 作业几乎一定是播种。Month Apr–May → op almost surely seeding.
若 / If 月份在 6–7 月 → 则 / then 作业为植保或深松。Month Jun–Jul → spray or deep tillage.
若 / If 月份在 9–11 月 → 则 / then 作业为收获。Month Sep–Nov → harvest.
若 / If 播种后约 30 天 → 则 / then 同设备同乡镇进入深松/植保窗口。~30 days after seeding → deep/spray window (same device×town).
若 / If 播种后约 5 个月 → 则 / then 同设备同乡镇进入收获（中位 152 天）。~5 months after seeding → harvest (median 152 days).
若 / If 乡镇有植保 → 则 / then 该乡镇具备播种+深松+收获全流程。Town has spray → full seed+deep+harvest pipeline.
若 / If 机手不是张志坚 → 则 / then 整季只绑定一台设备。Operator ≠ Zhang → exactly one device all season.
若 / If 同设备收/播面积比 > 3 → 则 / then 视为外向型代收为主。Device har/sow > 3 → treat as outward custom harvest.
若 / If 乡镇仅有收获 → 则 / then 视为代收/跨区服务目的地。Harvest-only town → custom / cross-area service destination.
若 / If 日期落在 8 月1日–9月25日 → 则 / then 档案中无任何作业记录（56天生长期空窗）。Date in Aug 1–Sep 25 → zero jobs (56-day growth silent window).
若 / If 设备属于全季面积前 5 名（共20台）→ 则 / then 这 5 台合计完成全季 54% 的作业面积。Device in top-5 by season area → those 5 do 54% of all area.
若 / If 组织为金丰公社 → 则 / then 植保面积占比≈32%，约为保农农场（17%）的两倍。Org = Jinfeng → spray area share ≈32%, ~2× Baonong (17%).
若 / If 同设备同作业类型 → 则 / then 幅宽整季几乎恒定（54/56 组仅一个档位）。Same device + same op → WIDTH nearly constant (54/56 groups single bin).
若 / If 机手确定 → 则 / then 所属组织唯一，两家组织人员互不交叉（11人 11机构，0 交叉）。Operator known → unique org; no staff overlap between the two orgs.
